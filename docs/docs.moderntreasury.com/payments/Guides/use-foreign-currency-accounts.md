# Using foreign currency accounts

# Overview

Most corporate banks allow you to hold foreign currencies in foreign currency bank accounts. For example, you could bank with JP Morgan and have three banks accounts in USD, EUR, and GBP. Holding a foreign currency allows you to send and receive money in that currency more easily, while also avoiding the need to exchange currencies often.

If you have foreign currency accounts that you'd like to connect to Modern Treasury, please ask your account manager to help you get set up.

Also see: [Originate a Global ACH payment](https://docs.moderntreasury.com/payments/docs/send-a-global-ach-payment)
