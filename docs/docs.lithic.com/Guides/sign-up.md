## Getting Started with the Lithic Dashboard

### Sign Up or Sign In

To access the Lithic Platform, you'll need to sign up for an account:

1. Visit the [Lithic Dashboard.](https://app.lithic.com)
2. Sign up for a new account or sign in if you already have one.

### First Steps After Signing In

Once you've signed into the Dashboard for the first time, here are some recommended actions to get started:

1. **Get an API Key** - Navigate to Program Settings > there you will see **Sandbox API Keys** for your API key. You'll need this to make API calls to the Lithic platform.

2. **Explore the Sandbox Environment** - Your account starts in Sandbox mode, which allows you to test all features without processing real transactions. Use this to familiarize yourself with the platform.

3. **Invite Users** - Go to Organization Settings to invite other users to your Sandbox account.

4. **Review Documentation** - Check out the API documentation and guides linked in the Dashboard to understand available features and integration options.

5. **Create a Test Card** - Try creating your first virtual card to see how card issuance works in the Lithic platform.

See: [quickstart](/docs/create-card) on how to use the Lithic APIs.
