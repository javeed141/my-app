# Incoming Payment Detail

In the sandbox, you may want to simulate workflows where you have received a payment in your bank account from an external party. To help with these simulations, we provide endpoints to simulate the creation of [Incoming Payment Details](https://docs.moderntreasury.com/platform/reference/incoming-payment-detail-object).

Related Guides: [Simulating an Incoming Payment Detail](https://docs.moderntreasury.com/platform/docs/simulating-an-incoming-payment-detail)
