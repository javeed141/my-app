# Payment Approval Rule Changes

For Payment Approval Rules, the following changes will require admin approval:

* New Payment Order approval rule creation
* Payment Order approval rule update
  * Updatable fields: name, description, conditions, and approvers
* Payment Order approval rule deletion

<br />

See our guide on Payment Order Approvals [here](https://docs.moderntreasury.com/payments/docs/create-an-approval-rule).
