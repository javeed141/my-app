# Create a New Release

Ready to create a new release for your repo? Simple.

Just click the **Working Copy** button and select **New Release**:

Autocode uses Semantic Versioning, so be sure to enter a valid semver express for the **Tag**.

Both **Title** and **Description** are optional but encouraged.
