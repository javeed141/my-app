# Site Navigation

All Site Navigation settings live under the **Navigation** tab in **Admin Settings**.

<Image align="center" border={false} src="https://files.readme.io/1cc70d1c0ef0313784f08366ef29e444e21ffec16cf16d52cbf8b650bfd60ec1-Navigation_settings.png" />

***

### Primary Navigation

#### Logo Link

By default, the logo links to your Landing Page. You can change this behavior by setting a custom homepage URL from **Admin Settings** → **Navigation**.

#### Landing Page

The Landing Page is the first page users see when they visit your documentation. You can set the Landing Page to one of the following options:

* Guides (default)
* Home
* API Reference

#### Left and Right Sections

By default, the Left Section contains only your logo, and the Right Section contains user controls. You can add items to either section and reorder them using the arrow controls.

### Subnavigation

Use Subnavigation settings to control which sections appear in your documentation and how they're labeled. You can:

* Show or hide the Version drop-down
* Show or hide entire page sections (Guides, API Reference, Recipes, and more)
* Rename page sections across your site using the drop-down controls
* Add custom pages or external links with custom titles

<Image align="center" border={false} src="https://files.readme.io/d526e7f83d89601ca65c4ed9978e61437dbe2d21b857f29de8e5518d4ec2d0d2-subnavigation.png" />

> 📘 Note
>
> Custom Pages cannot be disabled. If you haven't added any Custom Pages, they won't appear in the navigation, so there's nothing to disable.
