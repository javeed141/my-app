# Find brokers in your area

# Introduction

Connecting local real estate brokers to properties they represent can unlock strategic advantages in marketing, partnerships, and lead generation. Whether you're looking to identify active brokers in your city or analyze their transaction history, this guide shows how to leverage Datafiniti’s People and Property APIs to uncover brokers operating in your desired location.

## Searching broker via license number

Datafiniti's People data allows you to find broker via their official state license number. You can utilize the *licenses* field to filter for the license type and / or number. Lets breakdown searching for a specific DRE licensed broker.

```json
{
	"query": "licenses.value:\"02036937\" AND licenses.type:\"DRE\""
}
```

* licenses.value - This will search for the numeric value of the license number
* licenses.type - this will search for the abbreviated license type used for the brokerage.

## Locating broker records in people data

Datafiniti's People Data connects individuals in the real estate industry to specific property records they represent. Brokers are identified with the `propertiesRepresented` field. You can learn more about this in our People Data Schema. Here’s an example of where this data appears:

`propertiesRepresented` – A key generated by Datafiniti to link a broker to a specific property.

```json
{  
    "propertiesRepresented": [  
        "us/tx/kingwood/26870kingscrescentdr"  
    ]  
}
```

Once you have this key, you can cross-reference it in the property database to extract full details on the broker’s activity, listings, and associated companies.

## Search for brokers by location

Let’s begin by searching for brokers within a specific geographic area. We’ll look for brokers in Kingwood, TX, who are linked to a property listing.

```json
{  
  "query": "country:US AND province:TX AND city:kingwood AND propertiesRepresented:\*",  
  "num_records": 5  
}
```

This search will return brokers who are actively representing properties in Kingwood.

## Extract the propertiesRepresented key

From the people data response, capture the `propertiesRepresented` value. This key is what you’ll use to look up detailed property records in the next step.

Example:

```json
{  
    "firstName": "David",  
    "lastName": "Aldridge",  
    "jobTitle": "Realtor",  
    "phoneNumbers": ["+1 713-471-5247"],  
    "propertiesRepresented": ["us/tx/kingwood/26870kingscrescentdr"]  
}
```

## Find properties linked to the broker

Use the Property Data API to retrieve full listing details for the `propertiesRepresented` key, including context and broker information.

```json
{  
  "query": "keys:\"us/tx/kingwood/26870kingscrescentdr\"",  
  "num_records": 1  
}
```

Your response will include detailed listing information, including the broker’s name, company, transaction history, and listing status.

> 📘 Cross referenced searches
>
> The property search API require an active subscription. Check your [settings](https://portal.datafiniti.co/settings/billing/subscription) to ensure you have available credits.

## Find broker based on sold listing price range

Let say you are looking for a brokers that sell properties within a certain price range. You can use our `maxRecentSoldPrice` and / or `minRecentSoldPrice` to range out your prices on sold listings. For this example lets look for broker within Boulder, CO for broker that sold property over $750,000 to $1,250,000.

```json
{
  "query": "province:CO AND businessCategories:\"real estate\" AND maxRecentSoldPrice:<=1250000 AND minRecentSoldPrice:>=750000",
"num_records": 1 

}
```

# Paginating your search results

If you're expecting a large number of brokers or results from your query, you can use the offset parameter to paginate through your dataset. For example, the following queries will return two pages of results with 5 records per page:

Page 1 example:

```curl
POST <https://api.datafiniti.co/v4/people/paginate?page=1&limit=50>
```

Page 1 Query

```json
{  
  "query": "country:US AND province:TX AND city:kingwood AND propertiesRepresented:\*",  
  "num_records": 50
}
```

Page 2 example:

```json
POST <https://api.datafiniti.co/v4/people/paginate?page=2&limit=50>
```

Page 2 Query:

```json
{  
  "query": "country:US AND province:TX AND city:kingwood AND propertiesRepresented:\*",  
  "num_records": 50
}
```

Repeat this with increasing offset values to loop through all available pages in your dataset.
