# Getting notified about pending approvals

Go to the [Account Settings](https://app.moderntreasury.com/settings/my_settings) page if you wish to change your notification settings. There are a few types of notifications that you can receive. For example:

* Receive a notification every time an object is created that needs approval.
* Receive a summary email when I approve objects.
* Receive a daily digest before bank cutoff times with all objects pending approval.
