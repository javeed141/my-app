# Installation

We recommended you install `modern-treasury-js` via the  `@modern-treasury/modern-treasury-js` [NPM package](https://www.npmjs.com/package/@modern-treasury/modern-treasury-js).  You may also install `modern-treasury-js` by directly including the script.

```shell npm
npm install --save @modern-treasury/modern-treasury-js
```

```text yarn
yarn add @modern-treasury/modern-treasury-js
```

```javascript <script>
<script src="https://cdn.moderntreasury.com/js/v1/modern-treasury.js"></script>
```
