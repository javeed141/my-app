# Embeddable Flow

An `EmbeddableFlow` is a object that allows you to embed and manage our pre-built UIs in your application.  You can use `EmbeddableFlows` to embed [`Account Collection Flows`](https://docs.moderntreasury.com/reference/account-collection-flow-object) or [`Payment Flows`](https://docs.moderntreasury.com/reference/payment-flow-object).
