# Setting Up Google SAML

### Setting up SAML

1. Visit [admin.google.com](https://admin.google.com) and add a new SAML app by visiting Apps > Web and mobile apps > Add app > Add custom SAML app. Click the plus button and setup a custom SAML app

2. Enter a name and upload a logo for the SAML app. You can use this logo: [\<\<\<\<\<\<https://files.readme.io/61cbe90-ReadMe\_Logo-brandmark.png>>>>>>](https://files.readme.io/61cbe90-ReadMe_Logo-brandmark.png)

3. Enter the SSO URL and Entity ID from your ReadMe Dash

![](https://files.readme.io/00915e1-Screen_Shot_2019-12-06_at_6.04.14_PM.png "Screen Shot 2019-12-06 at 6.04.14 PM.png")

5. Setup Attribute Mappings

![](https://files.readme.io/d8d7dee-Screen_Shot_2019-12-06_at_6.01.10_PM.png "Screen Shot 2019-12-06 at 6.01.10 PM.png")

6. Click `Edit Service` and enable the app for everyone, or the subset of users that you want to have access to your ReadMe dashboard.

![](https://files.readme.io/33993c0-ss.png "ss.png")

You should be all set!
