# Address Normalization

Datafiniti normalizes all street addresses in our property database.  We convert common street designations to 2-3 character abbreviations.  These conversions are shown below.

## Property Search API Example

Let's say you want to search for the following address:

* 3407 Pennsylvania Avenue, Austin, TX 78721

In this case, we will need to abbreviate **suffix street name** `Avenue`to Datafiniti's normalization. For this example we will use the following property search API query to show how `Avenue`would be abbreviated to `Ave`. Below is what address your would need to use in the API query. For a full list of possible value Datafiniti normalizes [Possible Values for Property Fields](https://docs.datafiniti.co/docs/possible-values-for-property-fields)

```json
{
  "query":"address:\"3407 Pennsylvania Ave\" AND city:Austin AND province:TX AND postalCode:78721",
  "num_records":1
}
```
