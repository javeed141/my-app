# Create a Report

This guide outlines how to create a report.

### To create a Report

1. Log into Modern Treasury
2. Navigate to the [Reporting](https://app.moderntreasury.com/reporting/reports) section from the left sidebar
3. Click **Create New Report**, and select the type of report you wish to create (e.g. Balance Reports, Payment Orders, Transaction Line Items, etc.)
4. Begin by selecting one or several Internal Accounts to include in the report
5. Set the fields you wish to filter your data with
6. When the filters are set, click **Save**
7. Title your report, and proceed to the next page
   * ***optional:** Manage report access by individually assigning users with "Viewer" or "No Access" capabilities*
8. Click **Save** to finish creating the report
