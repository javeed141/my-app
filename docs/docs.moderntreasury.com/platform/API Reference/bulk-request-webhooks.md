# Bulk Request Webhooks

| Event          | Description                                                                                                   |
| :------------- | :------------------------------------------------------------------------------------------------------------ |
| **pending**    | The Bulk Request has been created and is queued for processing.                                               |
| **processing** | The Bulk Request is currently processing.                                                                     |
| **completed**  | The Bulk Request has been processed. You can check the output using the [Bulk Results](https://docs.moderntreasury.com/platform/reference/bulk-results) API. |
