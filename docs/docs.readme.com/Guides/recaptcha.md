# reCaptcha

Enabling reCaptcha within ReadMe will set up reCaptcha within the discussion forums. It will be implemented when a user comments on a discussion, or when someone creates a new post.

![](https://files.readme.io/4ffc5d0-Screen_Shot_2018-11-21_at_2.29.17_PM.png "Screen Shot 2018-11-21 at 2.29.17 PM.png")

To enable reCaptcha, head over to the [Google Admin panel](https://www.google.com/recaptcha/admin#list) and create a key pair. We support Google's Invisible reCaptcha v2, so ensure that the keys created are for this particular strategy.

![](https://files.readme.io/3fcce05-Screen_Shot_2018-11-21_at_2.20.31_PM.png "Screen Shot 2018-11-21 at 2.20.31 PM.png")

Once you've acquired the keys (site and secret), head over to the integrations page in the ReadMe dashboard, and set them. It's important that both keys be applied.

![](https://files.readme.io/622c3c3-Screen_Shot_2018-11-21_at_2.23.48_PM.png "Screen Shot 2018-11-21 at 2.23.48 PM.png")
