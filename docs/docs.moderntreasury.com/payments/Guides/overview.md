# Payments Overview

Companies use the **Payments** product to embed payments in their applications, orchestrate payments across currencies, and manage liquidity. Payments supports multiple rails and currencies via simple APIs.

We have created a set of guides to help you quickly get started and simulate even the most complex workflows and edge cases. Get started now by [sending a payment](https://docs.moderntreasury.com/payments/docs/quickstart).

[Get in touch](https://www.moderntreasury.com/talk-to-us) if you have any questions about using Payments.
