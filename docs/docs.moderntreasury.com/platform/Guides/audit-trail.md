# Audit Trail

The Modern Treasury Audit Trail is an **immutable, tamper‑resistant record of user actions taken in Modern Treasury**, captured with who performed the action, what changed, when it happened, and the affected object.

Audit trails span operational events such as payment creation, approval, modification, rejection, and transmission, as well as administrative changes like roles, permissions, and configuration updates. These logs are exportable from the dashboard and can be streamed to security tooling for monitoring and investigations. They exist to support compliance, internal controls, and forensic review by providing a trustworthy, third‑party source of truth for activity across the platform.

The full audit trail can be viewed by navigating to Settings > [Audit Trail](https://app.moderntreasury.com/settings/audit_records). Those without Administrator permissions will not be able to view the full audit trail.
