# Google Analytics

The Google Analytics integration lets you include your own Google Analytics on ReadMe. We support both [Universal Analytics](https://support.google.com/analytics/answer/2790010) and the more recent [Google Analytics 4](https://support.google.com/analytics/answer/10089681).

When you sign up for Google Analytics, you're given an ID (something like `UA-12345677` or `G-1234567ABC`). You can find it by going to your Google Analytics dashboard, and looking next to the site name.

Copy that code, go to your Project Settings on ReadMe (**Configuration > Integrations**), and enter it in as your Google Analytics ID.

This will start streaming basic analytics (i.e. page views) to your Google Analytics account.
