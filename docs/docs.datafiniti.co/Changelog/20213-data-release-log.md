# 2021.3 - Data Release Log

# Overview

Minor changes and updates with most focus on property data.

## Record Counts

The following are the updated record counts for each vertical:

* Business data decreased by 0.4 million to 117.8 million
* People data has remained at 3 million
* Product data increased by 4.2 million to 269.7 million
* Property data increased by 0.7 million to 117.7 million

## New Sources

The following data types have received new sources:

* 1 Property data source

## Schema Changes

* Property data now has `yearBuilt` as a top level field, with the data being extracted from `features`

## Fixes

* Prices data merging of merchants as been improved
* Updated 2 Product data sources and 4 Property data sources
