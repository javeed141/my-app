# Site Footer

> 📘 Available on Business and Enterprise Plans!
>
> White labeling options including removing the ReadMe logo and adding a customer footer are available only on Business and Enterprise plans. For plan details and feature comparisons, visit our <Anchor label="pricing page." target="_blank" href="https://readme.com/pricing">pricing page.</Anchor>

## Remove the ReadMe Logo

Go to **Admin Settings** → **Footer** to hide the ReadMe logo in your documentation footer.

## Custom Footer HTML

After you remove the ReadMe logo, you can add custom HTML to the footer. The footer accepts raw HTML and renders it across all documentation pages.
