# Create an API key

Creating an API key is the first step needed to make API calls. The steps listed below demonstrate the process of creating an API key.

# Create an API Key

1. Login to [Modern Treasury](https://app.moderntreasury.com/)
2. Verify the correct environment is being accessed
3. Navigate to the [Developers](https://app.moderntreasury.com/developers/api_keys) tab listed on the left panel, and then go to the API Keys section
4. Select the “Create API Key” button on the far right
5. Provide a name for the API key in the name field
6. Optionally, add “Allowed IPs” by clicking the add symbol and entering an IP address (The add option can be used multiple times to include an array of IP addresses)
7. Update the “Permissions” section with the desired level of access for each object (There is a question mark to the right of each object that provides further details)
8. Click on “Create” when the API Key is ready to be created
