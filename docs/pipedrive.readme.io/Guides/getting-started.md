# Getting started

> 📘
>
> Check out our [changelog](https://developers.pipedrive.com/changelog) for the latest changes regarding our Developer Platform.

We’re excited about the potential to connect. Pipedrive is a global CRM with headquarters in the USA and operations in the USA and EU, featuring an intuitive [RESTful API](https://developers.pipedrive.com/docs/api/v1/). You can use our API to create public or private apps using [OAuth 2.0](https://pipedrive.readme.io/docs/marketplace-oauth-authorization) and integrations via [API token](https://pipedrive.readme.io/docs/how-to-find-the-api-token). Public apps and integrations work directly with Pipedrive and are available in Pipedrive’s [Marketplace](https://www.pipedrive.com/en/marketplace).

## Extend your app's functionality

<hr />

[App extensions](https://pipedrive.readme.io/docs/app-extensions) allow apps to add interactive features that extend their app’s functionality in Pipedrive’s user interface.

Let users perform your app’s related actions through [link actions](https://pipedrive.readme.io/docs/app-extensions-actions). Allow users to see and use the data from your app in a panel with predefined components through [JSON panels](https://pipedrive.readme.io/docs/app-extensions-panels). Load any contextual web content in embedded iframes using our [custom UI extensions](https://pipedrive.readme.io/docs/custom-ui-extensions). Embed your app into a designated area in Pipedrive with our [manifest-based extensions](https://pipedrive.readme.io/docs/app-extensions#manifest-based-app-extensions).
