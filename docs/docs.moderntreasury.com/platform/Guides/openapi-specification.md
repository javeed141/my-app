# OpenAPI Specification

Modern Treasury also maintains [an OpenAPI specification](https://github.com/Modern-Treasury/modern-treasury-openapi) that allows you to use a wider set of developer tools such as [Postman](https://learning.postman.com/docs/integrations/available-integrations/working-with-openAPI/) or open source SDK generators.

# Postman

You can use Postman by importing Modern Treasury's OpenAPI Specification. You can choose to use a static version of the specification or keep it in sync with the repository to receive automatic updates. Learn how to import OpenAPI specifications into Postman by clicking [here](https://learning.postman.com/docs/integrations/available-integrations/working-with-openAPI/).
