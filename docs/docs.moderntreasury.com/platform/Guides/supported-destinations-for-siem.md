# Supported destinations for SIEM

Below are currently supported destinations. Additionally, data can be sent to a generic HTTP endpoint.

* Amazon S3
* Apache Iceberg Tables
* Amazon Redshift
* OpenSearch Service
* OpenSearch Serverless
* Datadog
* Honeycomb
* Coralogix
* Dynatrace
* LogicMonitor
* Logz.io
* MongoDB Cloud
* New Relic
* Snowflake
* Splunk
* Splunk Observability Cloud
* Sumo Logic
* Elastic

> 📘 Latest Supported Destinations
>
> Modern Treasury SIEM Integration utilizes [AWS Firehose](https://aws.amazon.com/firehose/) as the underlined technology. For the latest supported destinations, please see [AWS Firehose documentation](https://docs.aws.amazon.com/firehose/latest/dev/create-destination.html).
