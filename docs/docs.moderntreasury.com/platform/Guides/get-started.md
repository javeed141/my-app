# Signing up for an account

# Overview

In this tutorial, we will step through how to create an organization on Modern Treasury. We will then explore some of the suggested and common next steps once you have successfully signed up.

Every organization is provisioned a sandbox which can be used for testing and integration. The sandbox is fully functional, with feature parity to a production environment. We also provide sandbox-only functionality to help you simulate realistic workflows without risk. For example, you can simulate payment returns without moving any money.

# 1. Create an organization

You can register for Modern Treasury and gain access to a sandbox for the newly created organization. To do so, head over to [https://app.moderntreasury.com/sign\_up](https://app.moderntreasury.com/sign_up). You will be able to sign up, verify your email address, and create a new organization.

# 2. Invite your team

You can follow this guide to [invite other users to your team](https://docs.moderntreasury.com/platform/docs/user-invitations-overview).

# 3. Get started

The best way to explore Modern Treasury is by product. We've created a set of guides that make it easy to get started with each of our products:

* [Payments Quickstart](/payments/docs/quickstart)
* [Ledgers Quickstart](/ledgers/docs/quickstart)
