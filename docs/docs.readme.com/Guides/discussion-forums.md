# Discussion Forums

The Support forum allows users to interact with each other to answer questions and learn to use your API or documentation in new ways!

1. To ask a new question, click on the **Ask a Question** button. This requires the user to sign-in or sign-up for a ReadMe account.

<Image align="center" className="border" border={true} src="https://files.readme.io/55d847d-CleanShot_2022-10-17_at_17.27.00.png" />

2. Fill out the question form:

<Image align="center" className="border" border={true} src="https://files.readme.io/0b1a867-CleanShot_2022-10-17_at_17.37.12.png" />

The Discussions section in your developer hub features a version of our Markdown editor, allowing anyone submitting questions to include links, code samples, checklists, etc.

<Image align="center" className="border" border={true} src="https://files.readme.io/029faef-new_Editor_in_discussions.gif" />

3. Your new question shows up at the top of the selected category. Other users can then *upvote* a question, in the top left of the page or *respond* to the question, at the bottom of the page. *Users can also vote from the forum page*

<Image align="center" className="border" border={true} src="https://files.readme.io/91c39b1-CleanShot_2022-10-17_at_17.40.22.png" />

4. Admins can mark a question as solved when a good response has been posted.

<Image align="center" className="border" border={true} src="https://files.readme.io/679ed3a-CleanShot_2022-10-17_at_17.40.22_2.png" />

5. Questions that have been solved show with a **Answered!** box on the forum page.

<Image align="center" className="border" border={true} src="https://files.readme.io/8765d2e-CleanShot_2022-10-17_at_17.44.01.png" />

# Notifications

The Project Admins will get a notification on any new posts. You as the poster, will get a notification if anyone responds to your post. Anyone who responds to your post will get subsequent notifications on any new responses.
