# API Key Permissions

API Keys are granted Permissions on creation, but Permissions cannot be updated once the API Key is created.

# Creating an API Key

When creating an API key you have two options for permissions:

API Key with All Permissions, you can create key with access to all Modern Treasury current and future resources by creating the key and assigning it the API Manage All Role.

API Key with Restricted Permissions, you can create a key with access to select Modern Treasury resources, by assigning permissions individually. Note that because permissions cannot be updated if you would like to make changes you must create a new API key with the changed permissions and rotate it for the old one.
