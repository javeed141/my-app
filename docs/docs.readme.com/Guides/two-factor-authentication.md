# Two-Factor Authentication

> 👍 Highly Recommended
>
> ReadMe strongly recommends enabling two-factor authentication (2FA) for an additional layer of security on your account.

ReadMe supports two-factor authentication (2FA). When logging into ReadMe with 2FA enabled on your account, you'll be required to enter a code from an authenticator app in addition to your password. We support any authenticator app that uses the **TOTP (time-based, one-time password)** algorithm.

## Enabling Two-Factor Authentication

To enable 2FA, navigate to the <b><a href="https://dash.readme.io/user/edit" target="_blank">Edit profile</a></b> settings in the dashboard, locate the **Two Factor Auth** section, click **Enable**, and follow the instructions.

![](https://files.readme.io/7f6a3ca-Screenshot_2019-09-17_17.22.15_copy.png "Screenshot 2019-09-17 17.22.15 copy.png")

## Backup Codes

When setting up 2FA, you'll be prompted to save 10 backup codes. These **one-time-use** backup codes can be used to log you into ReadMe if you are unable to access 6-digit codes from your authenticator app.

> ⚠️ Keep your backup codes safe!
>
> Remember that these backup codes are **one-time-use**—so if you just have a few (or zero) remaining, you can generate new ones in the <a href="https://dash.readme.io/user/edit" target="_blank">profile settings</a> by clicking the **View Backup Codes** button. Generating new backup codes will invalidate your old ones!
>
> If you cannot access your authenticator app and you've lost your backup codes, please contact us at [support@readme.io](mailto:support@readme.io).

## Recommended Authenticator Apps

We recommend <a href="https://1password.com/" target="_blank">1Password</a>, which allows you to <a href="https://support.1password.com/one-time-passwords/" target="_blank">access one-time passwords</a> via a web browser extension, and desktop and mobile applications.

Other recommended apps include <a href="https://support.google.com/accounts/answer/1066447" target="_blank">Google Authenticator</a>, <a href="https://www.microsoft.com/en-us/account/authenticator" target="_blank">Microsoft Authenticator</a>, and <a href="https://authy.com/download/" target="_blank">Authy</a>.
