# How To

Export Specs, Schemas, Generators and more.

Autocode is completely hackable. Use **Exports** to export elements:

* [Specs](https://autocode.readme.io/docs/specs) - passes data to generators
* [Schemas](https://autocode.readme.io/docs/schemas) - validates Specifications
* [Generators](https://autocode.readme.io/docs/generators) - uses code templates and other Autocode Elements to generate data
* [Engines](https://autocode.readme.io/docs/engines) - template engines such as Handlebars, Mustache, Hogan, HAML and more
* [Helpers](https://autocode.readme.io/docs/helpers) - helpers used by template engines
* [Processors](https://autocode.readme.io/docs/processors) - manipulates Inputs before code generation
* [Transformers](https://autocode.readme.io/docs/transformers) - manipulates Outputs after code generation
* [Injectors](https://autocode.readme.io/docs/injectors) - inject code before code generation
