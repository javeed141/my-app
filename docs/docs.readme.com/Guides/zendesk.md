# Zendesk

Zendesk is a support platform, which offers a widget for your site. This integration will allow you to add the Zendesk support widget onto your ReadMe site.

# How to Integrate Zendesk With ReadMe

First, configure your web widget in [Zendesk](https://support.zendesk.com/hc/en-us/articles/203908456-Using-Web-Widget-to-embed-customer-service-in-your-website).

Next, in your Integration settings, add your Zendesk subdomain name. For example, for readmeio.zendesk.com you just need to put "readmeio." If it's enabled on your account, an icon will appear on the bottom right of your page.
