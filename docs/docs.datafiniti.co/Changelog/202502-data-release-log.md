# 2025.02 - Data Release Log

# New to Datafiniti

## New Supported Property Statuses

We’ve expanded our property data to include two new statuses, giving you more flexibility and accuracy in your searches:

* For Rent By Owner
* For Sale By Owner

These new statuses will allow you to identify properties sold or rented directly by owners, providing unique opportunities to connect with potential leads without a broker.

* forrentbyowner
* forsalebyowner

## New Property Data Schema Fields

We’ve added several new fields to our property data schema to provide more detailed information. These fields include:

* "companies.dateSeen": "Date"
* "companies.email": "String"
* "companies.equity": "Integer"
* "companies.equityPercent": "BigDecimal"
* "companies.landline": "Boolean"
* "companies.name": "String"
* "companies.phone": "String"
* "companies.title": "String"

Additionally, we’ve introduced a new field for people data:

* "people.landline": "Boolean"

These new fields will help you gain deeper insights into property ownership and company-related data, making it easier to connect with the right people and businesses companies.
