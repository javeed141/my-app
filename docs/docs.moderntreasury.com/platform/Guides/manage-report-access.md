# Manage Report access

There may be times when you need to hide data and limit access to specific users. As the owner of a report, you can share access or limit access to the report with other users in your organization.

> 🚧
>
> Sharing a report with users will give them access to all the data in that report, regardless of their role or permissions in Payments.
>
> For example, if you share a report on an Internal Account that the user does not have access to view, they will be able to view data for that Internal Account in the report. However, this does NOT give the user access to view or manage payments in the rest of the Modern Treasury platform.

## To manage Report access

1. Log into Modern Treasury
2. Navigate to the [Reporting](https://app.moderntreasury.com/reporting/reports) section from the left sidebar
3. Select the report you wish to manage access for
4. In the upper right corner, click on the ellipses (**...**) dropdown menu, and select **Manage Access**
5. Begin by searching for specific users and adjusting the permissions as needed. You can type in the name of a specific user or select from the dropdown menu
6. To remove a viewer, simply select **No Access** and save
7. Click **Save** to finish
