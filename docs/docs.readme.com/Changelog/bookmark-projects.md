# Bookmark Projects

<Image align="center" src="https://files.readme.io/b9538d3a419559081769a102deee592b3030cad4bcda5875fa8597f7ffd90018-Changelog_1.png" />

You can now bookmark projects. They'll stay pinned to the top of your project list page for as long as you'd like. :)

### New Features & Improvements

* **My Developers**: My Developers will now automatically filter to the currently viewed endpoint page <a class="badge" href="https://blog.readme.com/readme-refactored-new-editing-experience-bi-directional-syncing-with-github-2/">Refactored</a>
* **My Requests**: Added a filter to the API keys menu
* **API Reference**: Improved OpenAPI definition upload performance <a class="badge" href="https://blog.readme.com/readme-refactored-new-editing-experience-bi-directional-syncing-with-github-2/">Refactored</a>
* **Git Connection**: It now takes one less click to sync a repository <a class="badge" href="https://blog.readme.com/readme-refactored-new-editing-experience-bi-directional-syncing-with-github-2/">Refactored</a>
* **Owlbot AI**: Improved Owlbot’s ability to answer questions about APIs with large API definitions
