# Standard Options for Permission Sets and Roles

# Standard Permission Sets

Standard Permission Sets provide a bundled set of permissions that a user needs to use Modern Treasury. For example, a Reconciliation Permission Set provides users with all the necessary permissions to conduct reconciliation in the platform, e.g create and view Expected Payments, Create and View Transactions.

Other examples of Standard Permission Sets include:

* Invoicing Permission Set
* Developer Permission Set
* Payments Permission Set

# Standard Roles

Standard Roles are available for common use cases including Finance, Developers, and Administrators. Each role will contain Standard Permission Sets.
