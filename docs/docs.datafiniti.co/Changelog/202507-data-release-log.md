# 2025.07 - Data Release Log

# New people data fields focused on brokers

We’ve added new fields to help you better assess real estate professionals and their sales activity over time. These new fields provide a more complete picture of broker performance and sales trends across regions.

* `maxRecentSoldPrice `– Maximum most recently sold price by the broker
* `minRecentSoldPrice `– Minimum most recently sold price by the broker
* `salesCountAllTime `– Total number of properties sold by the broker
* `salesCountLastYear` – Properties sold by the broker in the last 365 days

<br />

# New property data fields

We’ve added the following sub-field to brokers to improve your reliability of searching brokers names within our property data. These fields are especially helpful for land evaluation and property investment comparisons.

* `capRate `– Capitalization rate of the property
* `capRateSourceURL `– Source link for the reported cap rate
* lotDepthValue – Numeric value of the lot’s depth
* lotDepthUnit – Unit of measurement for lot depth
* sewerType – Type of sewer access (public, private, septic)

# New business data sources

Our business data now includes data sourced from state governments to provide more complete coverage of local businesses. These buildouts are focused on expanding our coverage of registered business within a given state. This will provide you with accessible business details when searching. We currently include data from the following states and will be adding more soon. If there are any states you would like to see first, contact us <atsupport@datafiniti.co>.

* Alaska – commerce.alaska.gov
* Delaware – data.delaware.gov
* Kentucky – sosbes.sos.ky.gov
* Mississippi – corp.sos.ms.gov
* Pennsylvania – data.pa.gov

# Cleaner Performance Audit and Improvements

We have implemented regular, reliable process for property audits. These audits should happen each month. Priority fields are focused on property data to check are `mostRecentStatus`, `mostRecentPrice`, and `mostRecentRental `fields.
