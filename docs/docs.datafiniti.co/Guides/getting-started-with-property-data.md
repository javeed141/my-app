# Getting started with Property Data

Pick one of the following guides to walk you through how to use the Datafiniti API for Property Data. We offer the following coding library examples:

* [Postman](https://docs.datafiniti.co/docs/property-data-with-postman)
* [cURL](https://docs.datafiniti.co/docs/property-data-with-curl)
* [Node.js](https://docs.datafiniti.co/docs/property-data-with-nodejs)
* [PHP](https://docs.datafiniti.co/docs/property-data-with-php)
* [Python](https://docs.datafiniti.co/docs/property-data-with-python)
* [Wordpress via WPGetAPI plugin ](https://docs.datafiniti.co/docs/property-data-with-wpgetapi)

You can also access data through our [web portal](https://portal.datafiniti.co).
