# Deactivate an API key

When an API key is no longer needed, deactivating it ensures it can no longer be used. The steps listed below demonstrate the process of deactivating an API key.

## How to Deactivate an API Key

1. Login to [Modern Treasury](https://app.moderntreasury.com/)
2. Verify the correct environment is being accessed
3. Navigate to the [Developers](https://app.moderntreasury.com/developers/api_keys) tab listed on the left panel, and then go to the API Keys section
4. Once the page is loaded, click on the API key that needs to be deactivated
5. The drawer will load with the API key's name along the top
6. Click on the "Actions" button to the right of the API key's name and select "Deactivate"
7. A prompt will appear asking, "Are you sure you want to deactivate this API Key? This cannot be undone."
8. Select "Confirm" to complete the deactivation or "Cancel" to stop the process

## Frequently Asked Questions

**Can deactivated keys be viewed?**

Yes! Deactivated API keys can be viewed on the same page as active API keys by selecting the "Include Deactivated" option directly above the Organization ID.
