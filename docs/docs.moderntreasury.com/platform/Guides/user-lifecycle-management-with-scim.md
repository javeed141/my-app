# System for Cross-domain Identity Management (SCIM)

User Lifecycle Management with SCIM

Modern Treasury supports SCIM (System for Cross-domain Identity Management specification), a protocol for directory sync.

With SCIM, your identity provider is the source of truth for your organization’s user lifecycle management. Once you have directory sync set up, all user and group provisioning will be automatically managed through your identity provider.

This guide will walk you through how to configure SCIM for your organization.
