# Download outbound bank files

> 📘 "Accounts" Access Prerequisites
>
> Users with “Manage, Review, and Edit Access” to **Accounts** have the ability to view and download outbound files that have been sent to your bank from within our web app.

## Outbound Data

For file-based bank connections, Modern Treasury records outbound data between our web app and your bank. The term outbound data refers to the outbound files Modern Treasury sends to your bank to initiate various Transactions (i.e. Payment Orders, Returns, Reversals).

## To download an outbound file

1. Navigate to the [Outbound Data](https://app.moderntreasury.com/outbound_data) tab
2. Search for the desired batch file
3. Click on the desired file transfer to view details
4. Click "Download File" to download the file
