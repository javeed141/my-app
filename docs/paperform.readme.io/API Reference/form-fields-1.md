# Form Fields

Using these endpoints, you can retrieve all the fields for a form, retrieve a specific field, and update an existing field. In addition to the parameters that are common to all types of field, certain fields allow specific updates. These include:

## Calculations

Update the calculation for a [Calculation](https://paperform.co/help/articles/calculations-guide/) field.

## Choices

Update the options for a [Multiple Choice](https://paperform.co/help/articles/how-do-i-add-multiple-choice-questions-to-my-form/) field.
**Choice options are limited to 1000 per field.**

## Dropdown

Update the options for a [Dropdown](https://paperform.co/help/articles/dropdown-field/) field.
**Dropdown options are limited to 1000 per field.**

## Products

Update products for a [Product](https://paperform.co/help/articles/add-and-configure-products/) field.
**Products are limited to 100 per form.**

## Scale

Update the options for a \[Scale] (<https://paperform.co/help/articles/scale-field/>) field.
