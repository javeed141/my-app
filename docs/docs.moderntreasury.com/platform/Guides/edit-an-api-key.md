# Edit an API key

Editing an API key allows for updates as needed. The steps listed below demonstrate the process of editing an API key.

# How to Edit an API Key

1. Login to [Modern Treasury](https://app.moderntreasury.com/)
2. Verify the correct environment is being accessed
3. Navigate to the [Developers](https://app.moderntreasury.com/developers/api_keys) tab listed on the left panel, and then go to the API Keys section
4. Once the page is loaded, click on the API key that needs to be edited
5. The drawer will load with the API key's name along the top
6. Click on the "Actions" button to the right of the API key's name and select "Edit"
7. Update the “Permissions” section with the desired level of access for each object
8. Click on “Update” when completed
