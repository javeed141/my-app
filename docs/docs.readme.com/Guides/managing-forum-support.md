# Managing Forum Support

Admins will be notified via email (or [Slack](https://docs.readme.com/main/docs/slack), if configured) that a new support forum question has been received.

Questions from the community show up in the forum:

<Image className="border" border={true} src="https://files.readme.io/d966fa6-Screen_Shot_2021-11-03_at_1.53.48_PM.png" />

Community members can upvote questions that they also have. Other community members and Admins can answer a question by clicking on the question and responding at the bottom of the page.

<Image className="border" border={true} src="https://files.readme.io/290860c-Screen_Shot_2021-11-03_at_1.42.29_PM.png" />

If you're signed in as an Admin, you can also mark questions as **Answered**.

<Image className="border" border={true} src="https://files.readme.io/46339f0-Screen_Shot_2021-11-03_at_2.01.42_PM.png" />

Questions marked as Answered will appear on the main forum page with an Answered badge.

<Image className="border" border={true} src="https://files.readme.io/a23493d-Screen_Shot_2021-11-03_at_1.51.57_PM.png" />

# Notifications

The Project Admins will get an email notification on any new posts. You will only start getting notifications to replies if you as an admin reply to a post, otherwise only regular users that respond to a specific post will continue to get notifications after the initial post.

If you've [configured Slack notifications](https://docs.readme.com/main/docs/slack), you will only receive Slack notifications for new posts, but not subsequent replies.
