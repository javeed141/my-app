# Generate an API Key

All requests to our API require an API key. To generate an API key, you need to create an account on [lithic.com](https://app.lithic.com). Each account comes with a Sandbox API key.

A Production API key is available when you are enabled for a production environment.

## Sign up for a Lithic account

Create an account: [lithic.com/signup](https://app.lithic.com/signup)

> ⚠ ️You are responsible for all financial activity associated with your Production API key
>
> Our Sandbox environment is free to use. We **highly** recommend you develop and test your integration against the Sandbox environment (sandbox.lithic.com) before transitioning to the Production environment (api.lithic.com)

## Access your Sandbox API key

You can immediately make requests to the Sandbox environment at **sandbox.lithic.com**. Access your Sandbox API key from your account page: [app.lithic.com/settings](https://app.lithic.com/settings)

## Sandbox capabilities

Lithic's Sandbox environment provides a fully functional API environment for developing and testing your app.

The Sandbox supports all Lithic API endpoints and allows you to quickly build features and test functionality without the compliance overhead or financial risk of the Production environment.
