# Hosted Counterparty Onboarding

A Counterparty Invitation is a customizable experience you can use to easily onboard a counterparty and collect their bank account details for payment. The experience is fully hosted by Modern Treasury, on a moderntreasury.com URL.

There are two ways to instantiate a counterparty onboarding flow: by API or manually in the Modern Treasury dashboard when creating a counterparty.

The onboarding flow URL can be emailed to the counterparty by Modern Treasury. Or, we can return the unique URL to you via API, so that you can embed the URL in your app or redirect the user there yourself.

Additionally, you may want to redirect your customer back to your app upon completion of the onboarding flow. We support providing a redirect URL where the user is sent upon completion.

You can listen to webhook events to respond when users complete counterparty invitations.

Hosted Counterparty Onboarding is not available for stablecoin accounts at this time.
