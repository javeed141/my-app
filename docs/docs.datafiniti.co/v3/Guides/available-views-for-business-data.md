# Available Views for Business Data

The `view` parameter in the API determines which fields are returned by your data request and the order in which they're shown.  It also determines if multi-valued fields will flatten or expand their individual elements into multiple entries.

Note that each `view` is available in either `json` or `csv` format.  The `format` parameter you use will determine which one is used.

Available views for business data:

* [businesses\_all](https://docs.datafiniti.co/docs/businesses_all)
* [businesses\_all\_menusFlat](https://docs.datafiniti.co/docs/businesses_all_menusflat)
* [businesses\_all\_nested](https://docs.datafiniti.co/docs/businesses_all_nested)
* [businesses\_all\_nested\_no\_reviews](https://docs.datafiniti.co/docs/businesses_all_nested_no_reviews)
* [businesses\_all\_reviewsFlat](https://docs.datafiniti.co/docs/businesses_all_reviewsflat)
* [businesses\_basic](https://docs.datafiniti.co/docs/businesses_basic)
