# Property Data

## Property Data from the Portal

Here you'll find a step-by-step walkthroughs for maximizing Datafiniti's Property Data and API to fulfill your data needs. Let's get started!

## Navigating to Property Data

Once you log into your account, this will bring you to the Datafiniti Dashboard. From there you can use the left Navigation Bar to select searching different data types. In this case select "Property Data"

![](https://files.readme.io/7cb0627-image.png)

Once you click "Property Data" you will be brought to Datafiniti's property data dashboard

## Quick Example Searches

A new feature from customer request is our "Quick Example Search Selectors" in our property data dashboard. These search are pre-generated query of our customer's most frequent request. From Commercial, Residential, and many common use case, we made it "one-click" to get start using Datafiniti's property data.

Simply just click the box you wish to start with. For this example we will use the "Residential - Sold Last 30 Days" quick search:

![](https://files.readme.io/83b3a92-image.png)Which will populate results of property sold in the last 30 days like so:![](https://files.readme.io/a2d9ee2-image.png)

From here we can start a download of this data

## Property Data API Builder

You can also select the API tab to build a Property search query similar to the our API reference guide would allow you to build. You can reference the API reference guide on how to build a query. In most cases you can copy and paste your API search query into the Property Data API Builder and click Search

![](https://files.readme.io/3d79d24d02e1a1df21d68588120cb419ab23dc7b0284365429752f1843366332-image.png)

For help and tip on more complex querying techniques, you can find out more [here](https://developer.datafiniti.co/docs/constructing-product-queries).
