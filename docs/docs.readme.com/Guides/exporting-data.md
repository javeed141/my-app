# Exporting Data

Export your docs using our built-in features, tooling, and our API.

## Doc Files & PDF Export

<Image align="center" border={false} width="500px" src="https://files.readme.io/cde479bfc0ec46c97dfdd1ca787394a57e30fe065683df300603982488a4709d-Audio_Guides-20250729-1503042x.webp" />

You can export the files in your docs or as a PDF from the **Branch** menu. PDF exports only include Guides and Components used on those pages is downloaded via web browser. Exporting files (as a ZIP) is sent to your e-mail once complete. It includes OAS files, and does not include images—though the path to those images will be included and accessible. (View [sample PDF](https://drive.google.com/file/d/1FCU5ukSKIsNaOGY4-lVlc0enwVX9e1ll/view?usp=share_link))

## GitHub

Our [GitHub integration](https://docs.readme.com/main/docs/bi-directional-sync#/) allows you to sync and export your data to GitHub. The content stored there is the same as what is included when you export a branch—including all your doc versions.

***

## API

You can use our [API](https://docs.readme.com/main/reference/getting-started#/) to export data from your docs—including images, metrics, and Owlbot AI.
