# API Features

Datafiniti’s Business Data API provides robust tools for tools for discovering, analyzing, and leveraging active business data, contact information, and state public records. With flexible query capabilities and extensive data coverage, you can build highly specific searches tailored to your use case. Listed here are several features Datafiniti's business search API allows you to use for people data searches:

* [Count](https://docs.datafiniti.co/docs/business-count)
* [Geolocation](https://docs.datafiniti.co/docs/business-geolocation)
* [Pagination](https://docs.datafiniti.co/docs/business-pagination)
