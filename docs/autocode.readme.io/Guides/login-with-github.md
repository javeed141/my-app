## Which permissions do you use?

Autocode is fully integrated with GitHub. Upon signing up, you are asked for the following permissions:

### Personal user data

We have read-only access to your email for support, email notifications and newsletters.

### Repositories

We have read/write access to your public/private repos in order to create or load your existing repos.
