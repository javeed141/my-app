# Retrieve an API Key

1. Login to [Modern Treasury](https://app.moderntreasury.com/)
2. Verify the correct environment is being accessed
3. Navigate to the [Developers](https://app.moderntreasury.com/developers/api_keys) tab listed on the left panel, and then go to the API Keys section
4. You can search existing keys using the text field above the table.
5. The secret key can be copied directly by clicking into the API key and copying from the details table.
