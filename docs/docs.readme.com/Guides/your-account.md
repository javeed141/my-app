## Account Settings

Access your Account Settings directly from the main project dashboard, or open any project and navigate to **Admin Settings** → **User icon** → **Account Settings**.

<Image align="center" src="https://files.readme.io/13d5156f531a8456083eceddcbbd1b48a47eccd5ce239c4020a6f8128639240b-account_settings.png" />

<Image align="center" src="https://files.readme.io/fc5c407f6b397663d8787dee028319250b8eae887a027564542a88c9f2344731-account_settings_2.png" />

In Account Settings, you can update your personal preferences and profile details. These changes apply automatically to every project you’re part of.

<Image border={true} src="https://files.readme.io/e87921a-Editing_Your_Profile.jpg" className="border" />

## Cancel Your Subscription

In the legacy dashboard, scroll to the bottom of **General Settings > Projects Settings**.

<Image alt="Cancel Subscription to stop paying and revert your project to a trial." border={true} caption="Cancel Subscription to stop paying and revert your project to a trial." title="delete-project.png" src="https://files.readme.io/a3364e5-delete-project.png" />

Only the [Project Owner](https://docs.readme.com/main/docs/manage-team) has permission to do this. Project owners are also able to delete the project and all associated data.

> 🚧 Make Sure You're 100% Sure Before You Delete a Project!
>
> **Warning:** Deleting a project is irreversible, even by ReadMe Support, so be extra sure before deleting!
