# Wire

A traditional wire transfer goes from one bank or credit union to another using a network such as Fedwire or the Society for Worldwide Interbank Financial Telecommunication (SWIFT).  Due to a higher cost, typically wires are used for larger transactions or where a party wants the transaction to be irreversible.
