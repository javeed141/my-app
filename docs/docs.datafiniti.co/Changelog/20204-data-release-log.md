# 2020.4 - Data Release Log

# Overview

April signals the beginning of our transition to utilize a new field in searching for specific sources.  Also a large increase across all data sources.

## Record Counts

The following are the updated record counts for each vertical:

* Business data increased by 7.7 million to 107.1 million
* People data  increased by 1 million to 12.7 million
* Product data increased by 16.4 million to 185.2 million
* Property data increased by 11.2 million to 108.1 million

## New Sources

The following data types have received new sources:

* 1 Business data source
* 4 Product data sources

## Fixes

* A large portion of Home Depot data was removed due to issues with merging, new data will replace the problematic merged records
* Property type values have been consolidated
* Categories in our People data has been updated to reflect actual values versus SIC codes
* Improved measurement unit normalization
* Updated 2 Product data sources
