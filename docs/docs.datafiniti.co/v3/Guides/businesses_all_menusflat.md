# businesses_all_menusFlat

The `businesses_all_menusFlat` view will display all available fields in our [business schema](https://datafiniti-api.readme.io/v3/docs/business-data-schema).

If using the `csv` format, it will split out elements in the `menus` field into separate rows.  This means a single business with multiple menu items will show up in multiple, contiguous rows while its each entry in its `menus` field will show up in a separate row.
